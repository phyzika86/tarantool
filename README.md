# tarantool
1. Запустить docker-compose up -d
2. Выполнится lua скрипт, который выполнит вставку тестовых данных в таблицу
3. Запустить скрипт app_test.py для проверки подключения
4. Запустить скрипт app_lessons.py для выполнения задания
5. Результаты выполнения запросов предоставлены в файле скрина
